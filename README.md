# BM Attorneys Website

A professional website developed for **BM Attorneys** that demonstrates the complete software development lifecycle, from requirements analysis and system design to implementation, automation, and testing.

## Live Website

🌐 https://b-malulekaattorneys.netlify.app

---

## Project Overview

This project was developed to provide BM Attorneys with a modern, responsive website while demonstrating industry-standard software engineering and business analysis practices.

The project includes:

- Software Requirements Specification (SRS)
- UML Activity Diagram
- Responsive Website
- End-to-End Testing using Playwright
- API Testing using Postman
- Workflow Automation using Make.com
- Test Evidence and Documentation
